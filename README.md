📌 Acceso  Datos

Este repositorio contiene dos temas principales relacionados con el acceso a bases de datos relacionales:

📂 Estructura del Repositorio

📁 Tema 1: Conectores para Bases de Datos Relacionales

En esta carpeta se encuentran ejemplos y configuraciones para conectar a bases de datos relacionales utilizando conectores específicos.


📁 Tema 2: Acceso mediante ORM (Object-Relational Mapping)

Esta carpeta contiene ejemplos y explicaciones sobre el uso de ORM para interactuar con bases de datos relacionales de manera más sencilla y estructurada.

📌 Tecnologías y herramientas utilizadas:

Hibernate (Java)
